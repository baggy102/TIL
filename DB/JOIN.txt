Inner Join

두 테이블 간 중복 값만 추출

Ex. 
SELECT * FROM IDOL_GROUP;
SELECT * FROM IDOL_MEMBER;

SELECT * FROM IDOL_GROUP A, IDOL_MEMBER B WHERE A.GROUP_NAME = B.GROUP_NAME;

(이 때 A, B 는 alias; 즉, ~~테이블을 앞으로 A라고 부르겠다.)

-> 일반적으로 위 예시 보다는 필요한 컬럼만 추출해서 사용

SELECT A.COMPANY, A.GROUP_NAME, B.MEMBER_NAME, B.REAL_NAME 
	FROM IDOL_GROUP A, IDOL_MEMBER B WHERE A.GROUP_NAME = B.GROUP_NAME;

만약 각 그룹에 몇명의 멤버가 있는지 보고싶다?

SELECT A.COMPANY, A.GROUP_NAME, COUNT(B.MEMBER_NAME) COUNT
	FROM IDOL_GROUP A, IDOL_MEMBER B WHERE A.GROUP_NAME = B.GROUP_NAME GROUP BY A.COMPANY, A.GROUP_NAME;



OUTER JOIN

기준 테이블으로 다른 테이블을 아우터 조인

EX.
A 를 기준으로 B를 아우터 조인

SELECT * FROM IDOL_GROUP A, IDOL_MEMBER B WHERE A.GROUP_NAME = B.GROUP_NAME(+)

동일한 결과를 LEFT OUTER JOIN 으로

SELECT * FROM IDOL_GROUP A LEFT OUTER JOIN IDOL_MEMBER B ON A.GROUP_NAME = B.GROUP_NAME;

(LEFT OUTER JOIN, RIGHT OUTER JOIN 유사한 기능, 일반적으로 LEFT 사용 혼동 방지)
